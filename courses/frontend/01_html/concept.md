# HTML 概念篇

> 万丈高楼平地起，HTML就是那地基

## 什么是HTML？

**HTML = 超文本标记语言**

用人话说：它是一种用来**描述网页长什么样**的语言。

- 你看到的每个网页，背后都是HTML
- 浏览器（Chrome、Safari等）负责把HTML"翻译"成你看到的页面

## 核心概念

### 1. 标签（Tag）

HTML用**标签**来标记内容：

```html
<h1>这是一个标题</h1>
<p>这是一段文字</p>
<a href="https://baidu.com">这是一个链接</a>
<img src="图片地址" />
```

规则：
- 大多数标签是**成对**的：`<开始标签>内容</结束标签>`
- 少数标签**自闭合**：`<img />`、`<br />`

### 2. 常用标签家族

| 标签 | 用途 | 例子 |
|------|------|------|
| `h1`~`h6` | 标题 | `<h1>大标题</h1>` |
| `p` | 段落 | `<p>一段话</p>` |
| `a` | 链接 | `<a href="网址">点击</a>` |
| `img` | 图片 | `<img src="地址" />` |
| `div` | 盒子（无意义） | `<div>容器</div>` |
| `span` | 小容器（无意义） | `<span>一小段</span>` |
| `ul/ol` | 无序/有序列表 | `<li>列表项</li>` |
| `table` | 表格 | `<tr>行<td>列</td></tr>` |
| `form` | 表单 | `<input /><button>` |

### 3. 属性（Attribute）

标签可以带**属性**，提供额外信息：

```html
<a href="https://baidu.com" target="_blank">打开新窗口</a>
<img src="logo.png" alt="公司Logo" width="100">
<input type="text" placeholder="请输入名字">
```

格式：`属性名="属性值"`

### 4. 嵌套（Nest）

标签可以**套标签**：

```html
<div>
  <h1>标题</h1>
  <p>内容<a href="#">链接</a></p>
</div>
```

**注意**：要正确嵌套，不能交叉！

```html
<!-- ✅ 正确 -->
<div><p>内容</p></div>

<!-- ❌ 错误 -->
<div><p>内容</div></p>
```

### 5. 语义化（Semantic）

用对的标签做对的事：

| 标签 | 语义 |
|------|------|
| `<header>` | 头部 |
| `<nav>` | 导航 |
| `<main>` | 主体 |
| `<article>` | 文章 |
| `<section>` | 章节 |
| `<footer>` | 底部 |
| `<aside>` | 侧边栏 |

**为什么重要？**
- SEO（搜索引擎更喜欢）
- 可访问性（屏幕阅读器更好用）
- 代码可读性（别人更容易看懂）

## 最小化必须掌握

| 优先级 | 标签 | 用途 |
|--------|------|------|
| ⭐⭐⭐⭐⭐ | div, h1-h6, p, a, img | 最常用 |
| ⭐⭐⭐⭐ | ul, ol, li, input, button | 表单和列表 |
| ⭐⭐⭐ | header, nav, main, footer | 语义化 |
| ⭐⭐ | table, form | 表格和表单 |
| ⭐ | span, br, hr | 辅助 |

## 常见误区

1. **❌ 什么都是div**
   - ✅ 用语义化标签：`<nav>`, `<header>`, `<article>`

2. **❌ 标签大写**
   - ✅ 小写：`<div>` 不是 `<DIV>`

3. **❌ 不写alt**
   - ✅ 加alt：`<img alt="图片描述">`

4. **❌ 忘记闭合**
   - ✅ 检查：每个`<div>`都有`</div>`
